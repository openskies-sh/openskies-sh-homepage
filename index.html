If you are involved in the development of the future aviation systems, we can help you by providing technical expertise, software development capability & project management services. To see some of the work we have done, take a look at our products and services below.



our products

Flight Passport

Flight passport is a OAuth Provider developed specifically for U-Space / UTM. Using this opensource project, you can be deployed it on any cloud service (national or international) and issue tokens and credentials to enable identity and authentication services for the U-Space / UTM eco-system. See Github page

Flight Spotlight

Flight Spotlight is a frontend applicaiton that helps you connect to a Discovery and Syncronization service instance. Using Flight Spotlight you can connect to the local DSS instance see all declared flights in a area. This open source tool can be deployed to any cloud (public or private). See Github page

Registry System

We are experts in drone registration, registry systems and bringing registries online. We have built and developed tools to securely bring registry online and make it interoperable with other registration systems. In addition to the API specification, we help in all aspects of regitry data and inter-operability. Github page.
