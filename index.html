our products

Open-UTM

The open-utm stack consists of Flight Spotlight (frontend) and Flight Blender (backend). Using Flight Spotlight you can connect to UTM / U-Space for standards compatible Network Remote-ID and link to ADS-B and other surveillance equipment for traffic information. Flight Blender implements modern UTM standards for data processing and exchange. See homepage

Aerobridge

Aerobridge is a management server for drone manufacturers and operators specifically with a focus on security. Using Aerobridge you can get build world class secure operations out of the box. See home page
